Indonesian Stock Market Prediction API

API endpoints for accessing stock market data

API Documentation
Available Endpoints:
  • /api/stocks - Get list of available stocks
  • /api/stocks/details - Get detailed list of stocks
  • /api/stocks/sectors - Get stocks grouped by sectors
  • /api/stock/{ticker} - Get historical data for a ticker
  • /api/technical/signal/{ticker} - Get technical signals for a ticker
  • /api/stocks/db - Get stocks from database
  • /api/stock/{ticker}/db - Get stock data from database
Database Operations:
  • /api/stock/refresh/{ticker} - Refresh data for a specific stock
  • /api/stocks/refresh/all - Refresh data for all stocks
